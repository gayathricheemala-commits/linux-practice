# Linux Practice Notes

This repository contains my Linux practice commands and basic cybersecurity usage.

## Basic Commands

pwd
Shows present working directory

cd /
Go to root directory

ls
List files and folders

mkdir test
Create directory

rm file.txt
Delete file

clear
Clear terminal

## Navigation Commands

cd ..
Move one directory back

cd ~
Go to home directory

## File Commands

touch file.txt
Create file

cat file.txt
View file content

cp file.txt newfile.txt
Copy file

mv file.txt /home/kali
Move file

## Permissions

chmod 777 file.txt
Change permissions
